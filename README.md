# Django Learning Project 🚀

This repository contains a Django learning project built while practicing Django fundamentals and backend development concepts.  
It demonstrates real-world usage of Django with multiple apps, templates, static files, authentication, and CRUD operations.

## ✨ Features
- User authentication (login, register, logout, profile)
- CRUD operations using Django ORM
- Django Forms and form validation
- Template inheritance
- Static files (CSS) and templates
- Multiple Django apps (Todo, Users, etc.)
- SQLite database for local development

## 🛠️ Tech Stack

- Python
- Django
- HTML
- CSS
- SQLite (development database)

## ▶️ Run Locally
git clone https://github.com/Shubham37204/Django.git

cd Django
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver

- Practice backend development using Django
